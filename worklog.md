## Режимы игры

### 1. Одиночная игра (Single Player)
- Доступна сразу на главной странице
- 2D реализация с CSS анимациями
- Тап по блокам для сбора
- Очки, комбо, промокоды

### 2. Мультиплеер (Projector + Phone)
- **Проектор**: `/projector` - 3D сцена с Three.js
- **Телефон**: `/phone` - Контроллер с кнопкой
- Связь через Socket.io сервер (порт 3001)

## Запуск в локальной сети

### 1. Запуск сервера
```bash
cd mini-services/game-server
bun run dev
```
Сервер запускается на порту 3001.

### 2. Запуск Next.js
```bash
bun run dev
```
Сайт доступен на порту 3000.

### 3. Подключение устройств

**На проекторе:**
```
http://[IP-СЕРВЕРА]:3000/projector
```

**На телефоне:**
```
http://[IP-СЕРВЕРА]:3000/phone
```

### 4. Игровой процесс
1. Открыть `/projector` на большом экране
2. Открыть `/phone` на телефоне
3. На проекторе нажать "НАЧАТЬ ИГРУ"
4. Тапать на телефоне для ломания блоков!

## Технологии

| Компонент | Технология |
|-----------|------------|
| Frontend | Next.js 16, React, TypeScript |
| 3D | Three.js |
| Стиль | Tailwind CSS, shadcn/ui |
| Real-time | Socket.io |
| Шрифт | Press Start 2P (pixel font) |

## События Socket.io

| Событие | Направление | Описание |
|---------|-------------|----------|
| `identify` | Client → Server | Идентификация (projector/phone) |
| `BREAK_BLOCK` | Phone → Server | Ломать блок |
| `block_destroyed` | Server → Projector | Блок разрушен |
| `update_score` | Server → Phone | Обновление счёта |
| `block_missed` | Projector → Server | Блок упал за линию |
| `start_game` | Projector → Server | Начать игру |
